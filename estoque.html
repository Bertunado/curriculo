Sistema de Gerenciamento de Estoque

Uma solução web para controle de inventário com Python e Django.

Visão Geral do Projeto

Este projeto foi desenvolvido para atender à necessidade de um controle de estoque eficiente em ambiente empresarial. A aplicação web, construída com o framework Django, oferece uma interface intuitiva para gerenciar produtos, registrar entradas e saídas, e manter um histórico preciso do inventário. A solução foi implantada na nuvem utilizando PythonAnywhere, garantindo acesso remoto e seguro.

Funcionalidades Principais

  • Cadastro de Produtos: Permite adicionar novos itens ao estoque com informações detalhadas.
  • Controle de Entrada e Saída: Interface para registrar facilmente a movimentação de produtos.
  • Listagem e Busca: Visualização completa do estoque com filtros e sistema de busca.
  • Interface Administrativa: Painel de controle robusto (Django Admin) para gerenciamento avançado.
  • Implantação na Nuvem: Acesso de qualquer lugar graças ao deploy no PythonAnywhere.

Tecnologias Utilizadas

Python Django SQL (SQLite) HTML CSS (Tailwind CSS) PythonAnywhere
