Robô de Day Trade para Criptomoedas

Automatizando a negociação de ativos digitais com Python e a API da Binance.

Visão Geral do Projeto

Este projeto consiste em um bot de "scalping" desenvolvido em Python para operar no mercado de criptomoedas. O robô se conecta à API da exchange Binance para monitorar o par BTC/USDT em tempo real, executando ordens de compra e venda automaticamente com base em uma estratégia predefinida. O objetivo é capitalizar sobre pequenas flutuações de preço para gerar lucro de forma automatizada.

Funcionalidades e Etapas

  • Conexão com a Exchange: Utilização da biblioteca CCXT para uma integração segura e robusta com a API da Binance.
  • Monitoramento em Tempo Real: Coleta contínua de dados do livro de ordens (order book) para o par BTC/USDT.
  • Lógica de Trading: Implementação de uma estratégia de scalping para identificar oportunidades de negociação de curto prazo.
  • Execução de Ordens: Capacidade de enviar ordens de compra (buy) e venda (sell) de forma programática.
  • Gerenciamento de Risco: Lógica básica para limitar perdas e realizar lucros.

Tecnologias Utilizadas

Python CCXT Library Binance API Websockets
