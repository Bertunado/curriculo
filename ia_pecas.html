Análise Preditiva de Vida Útil de Peças

Utilizando Inteligência Artificial para otimizar a manutenção industrial.

Visão Geral do Projeto

Este projeto de Machine Learning foi desenvolvido para prever a vida útil de componentes industriais com base em dados históricos de operação e falhas. O objetivo principal é otimizar os ciclos de manutenção, substituindo a abordagem reativa (corrigir após a falha) por uma estratégia preditiva, o que resulta na redução de custos e aumento da eficiência operacional.

Funcionalidades e Etapas

  • Análise Exploratória de Dados: Tratamento e visualização dos dados para identificar padrões e correlações.
  • Engenharia de Features: Criação de variáveis relevantes para treinar o modelo de Machine Learning.
  • Treinamento do Modelo: Implementação e treinamento de um modelo de classificação para prever a probabilidade de falha.
  • Validação e Métricas: Avaliação da performance do modelo utilizando métricas como acurácia, precisão e recall.
  • Interface de Simulação: Desenvolvimento de uma interface simples para que o usuário pudesse inserir dados e receber a previsão de vida útil.

Tecnologias Utilizadas

Python Pandas Scikit-learn Matplotlib / Seaborn Jupyter Notebook Flask (para a interface)
