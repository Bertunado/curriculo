Visão Geral de Máquinas da Fábrica

Sistema interativo para visualização de peças e gestão administrativa.

Sugestão: Substitua por um print da tela de visualização das máquinas.

Visão Geral do Projeto

Este foi um dos meus primeiros projetos utilizando Django e serviu como uma solução para a visualização do parque fabril. A aplicação permite que qualquer usuário navegue por uma lista de máquinas e, ao clicar em uma delas, veja todas as peças correspondentes. A principal funcionalidade é o sistema de autenticação, que concede a superusuários a capacidade de adicionar novas peças diretamente pela interface do site, sem precisar de acesso ao código ou ao banco de dados.

Funcionalidades Principais

  • Visualização Interativa: Interface onde o usuário clica em uma máquina para listar suas peças.
  • Sistema de Autenticação: Mecanismo de login seguro para diferenciar usuários comuns de administradores.
  • Painel de Superusuário: Área restrita onde administradores podem adicionar, editar e remover peças.
  • Gerenciamento via Web: Toda a gestão de peças é feita diretamente pelo site, garantindo agilidade.
  • Estrutura com Django: Utilização do poder do Django para criar um sistema robusto e escalável.

Tecnologias Utilizadas

Python Django SQL (SQLite) HTML CSS Sistema de Autenticação Django
